## Summary : 
This application will allow the user to get information regarding wind speed & weather
information of a specific predefined spots pinned on a map, there shall be an option to view a "report" for
the specific location in a specific time-frame on a different view.

The application will allow the user to sign up / log in and to setup a wind speed threshold for a specific
predefined spot which he will then get notified when the threshold is hit.

The application consists both the frontend application and a backend node.js server.
Frontend application will handle the data presentation (data and notifications) and inputs from the user.
Backend server will handle the relevant API calls and threshold checks & notifications.
